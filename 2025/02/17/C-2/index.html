C++简单程序设计

这个专题主要是用于已经有C语言基础的，把基本的语法部分过一过

C++特点

  • 兼容C（支持面向过程的方法）

  • 支持面向对象的方法

  • 支持泛型程序设计方法[例：不论什么数据类型都可以用这个程序进行排序，即将数据类型进行抽象，更加通用]

对象object：“即帮忙干事的人，我把东西给你，你进行一系列的处理”“常用来做一些常见的，自己做起来很麻烦的活”

3.词法记号

关键字：C++预定义的单词，不能作为自用

文字：程序直接使用符号表示的数据

4.标识符的构成规则

1.以大小写字母或下划线_开始

2.大小写字母表示的标识符不同

5.基本数据类型

C++能够处理的基本数据类型：整数、浮点数、字符、布尔

数据中的数据：常量[源程序中直接写明的数据，在运行期间不可更改]，变量

ps：C++基本数据类型中没有字符串类型，标准库中有string类型。

5.1符号常量（由const修饰的对象）

定义常量语句

常量表达式是一类值不能发生改变的表达式，其值在程序编译阶段确立，便于程序优化

ps:文字表达式与由常量表达式初始化的const对象都是常量表达式

const float PI=3.14; or float const PI=3.14

constexpr修饰的对象暗含了const属性，并且必须由常量表达式初始化constexpr int size=get_size();

//要求getsize()为常量表达式，即函数返回常量表达式

const int size=get_size()不是常量表达式

const int max_size=100; const int limit=max_size+1 两个都是常量表达式

3.浮点数：float,double,long double（表示为12.5；或者0.345E+2(10的二次方)指数形式）

ps:浮点常数默认为double类型，可以在后缀加上F使其变为float型，即2.1f

8.逗号运算：一般用于for循环中，两个自增

i = (j++, j+100, 999+j);

j 最初的值为 10，然后自增到 11，这时 j 的值就为 11，然后再计算第二个表达式 j+100，最后 j（j 为 11） 再加上 999，最后 i 为 最后一个表达式的值 999+j，及 999+ 11=1010。

10.逻辑运算与逻辑表达式

优先级依次降低 !非 &&与 ||或； 表达式写为 ()&&()

ps:&&具有”短路特性“，用于(i!=9)&&(++j!=9)，如果i!=9满足，那么++j!=9就会进行判断了，导致此时j并不会+1 （||类似） 所以比较大小就比较别顺便改变变量值了，避免这种错误。
