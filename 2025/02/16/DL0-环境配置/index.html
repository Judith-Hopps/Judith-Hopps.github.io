关于查找显卡信息：

右键点击任务栏->选择[任务管理器(K)]->选择[性能]

可以看到显示有CPU GPU0 GPU1等信息。

GPU 0：intel(R)即属于因特尔公司生产 Graphics即集成显卡[也就意味这种GPU一般是属于继承在CPU内部的，并不属于集成显卡，不是深度学习所需要的。如果仅有该GPU，那么查询时nvcc -V则显示没有该指令]

GPU 1：NVIDIA即属于英伟达公司生成 GeForce即隶属于独立显卡GeForce系列。[这就是实验需要的显卡，如果没有这个GPU，则相关的配置就先不配置了，即使没有它，暂时也能完成前面部分学习]

总的来说

Nvidia GeForce GT 630是独立显卡。

Nvidia即英伟达，凡是显卡型号带有Nvidia的都是独立显卡

Intel不制造独立显卡，凡是显卡型号前面带有intel的都是英特尔的集成显卡。

AMD的独立显卡包括品牌 ATI 和 Radeon，其他的是集成显卡。

更多内容：为什么做GPU计算，深度学习用amd显卡的很少，基本都nvidia？

    TF_ENABLE_ONEDNN_OPTS环境变量控制TensorFlow是否使用oneDNN来加速某些操作，如卷积、池化等。当设置为0时，表示禁用oneDNN的优化。

    TensorFlow binary is optimized to use available CPU instructions in performance-critical operations.

    这个警告是 TensorFlow 在启动时输出的信息，提醒用户一些当前 TensorFlow 二进制文件已经经过优化，以利用充分可用的 CPU 指令集执行性能关键的操作。
